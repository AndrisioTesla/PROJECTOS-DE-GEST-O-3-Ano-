
## 📌 Sobre o Projeto

O TruthGuard AI é uma plataforma inteligente de deteção de Fake News desenvolvida em Python com técnicas de Inteligência Artificial e Processamento de Linguagem Natural (NLP).

O sistema permite que utilizadores insiram notícias ou textos suspeitos e recebam uma análise automática indicando a probabilidade de a informação ser verdadeira ou falsa.

O objetivo do projeto é contribuir para o combate à desinformação digital, promovendo uma sociedade mais informada e consciente.
